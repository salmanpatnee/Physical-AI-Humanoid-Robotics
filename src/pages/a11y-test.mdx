## Testing Instructions

To test this page for accessibility:

1. **Visual Review**: Switch between light and dark modes using the theme toggle
2. **Contrast Check**: Run `npm run test:contrast` to verify WCAG AA compliance
3. **Screen Reader**: Test with a screen reader (NVDA, JAWS, or VoiceOver)
4. **Keyboard Navigation**: Navigate the page using only the Tab key
5. **Zoom Test**: Zoom to 200% and verify readability

All text should be clearly readable in both light and dark modes with sufficient contrast ratios (minimum 4.5:1 for normal text, 3:1 for large text).
