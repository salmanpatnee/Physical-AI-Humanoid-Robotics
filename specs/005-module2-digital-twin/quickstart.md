# Quickstart: Module 2 - The Digital Twin (Gazebo & Unity)

This document provides a quick overview of how to get started with the new module.

## Prerequisites

-   You have a local development environment set up for the Docusaurus project.
-   You have Gazebo and Unity installed on your system.

## Running the Simulation

1.  Navigate to the `examples/module2` directory.
2.  Follow the instructions in the `README.md` file to launch the Gazebo and Unity simulations.

## Viewing the Content

1.  Start the Docusaurus development server:
    ```bash
    npm run start
    ```
2.  Open your web browser and navigate to `http://localhost:3000`.
3.  The new module, "The Digital Twin (Gazebo & Unity)", will be available in the sidebar.
