# Quickstart: Viewing the Weekly Course Schedule

This guide explains how to access and view the new Weekly Course Schedule page.

## Prerequisites

- The Docusaurus project must be running locally.

## Steps

1.  **Start the Docusaurus Development Server**.
    - Open a terminal in the project root directory.
    - Run the command `npm start`.
    - Wait for the build to complete and the local server to start.

2.  **Navigate to the Course Schedule Page**.
    - Open your web browser and go to the local Docusaurus URL (usually `http://localhost:3000`).
    - In the sidebar on the left, look for a new link titled "Course Structure" or "Weekly Schedule". It will be located directly after the "Introduction" section and before "Module 1".
    - Click on this link.

3.  **View the Schedule**.
    - The page will display the full 13-week course schedule.
    - You can view the topics, activities, and time estimates for each week.
    - A visual timeline at the top of the page will show how the weeks map to the four main course modules.
    - Clickable links within the weekly topics will take you directly to the relevant chapter pages.
